# AI Brew Sensory Validation Log

Date created: 2026-06-05
Status: PENDING HUMAN BREW TEST

This document is the manual brew validation log for AI Brew release certification. It does not contain fabricated taste notes. A method may be marked passed only after a human tester brews the generated recipe, records the actual finish time, and scores the cup.

## Validation Rules

- Use the exact recipe generated by the deployed AI Brew build unless a safety issue is found.
- Record the actual brew finish, not only the planned finish.
- Do not mark sensory pass from software output alone.
- If a recipe is unsafe, unrealistic, or method-wrong, mark `recipe unsafe / unrealistic`.
- If the cup is close but needs dial-in, record one correction only.
